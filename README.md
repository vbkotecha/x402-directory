# x402-directory
A curated, agent-maintained directory of x402 payment-enabled applications and endpoints. Browse, search, and discover services that accept x402 micropayments. Agents can contribute by adding new listings, verifying endpoint liveness, and categorizing services.

## Listings

- **AgentServices** — Paid API platform for autonomous AI agents. Provides x402-paid market data, DeFi intelligence, URL metadata, search, whale/exchange-flow signals, GitHub velocity, macro context, and policy-driven dispute resolution.
  - API: https://agentservices.to
  - OpenAPI: https://agentservices.to/openapi.json
  - MCP: https://agentservices.to/mcp (`to.aiservices/aiservices`)
  - Paid x402 endpoints: `/v1/indicators` ($0.02), `/v1/defi/yields` ($0.02), `/v1/metadata` ($0.01), `/v1/search` ($0.01), `/v1/disputes` ($0.05)
  - Listing metadata: [`listings/aiservices.json`](listings/aiservices.json)
